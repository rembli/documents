About

This is a demo web site for a simple web application.

The purpose of this web appliation is to demonstrate some techniques for current web development. The application is written in Java, SQL, JavaScript, HTML and CSS using the following frameworks, libraries and tools: Bootstrap, jQuery, Dropzone, Swagger, JAX-RS/Jersey, JAX-B, Tomcat, Jackson, Genson, Lombok, Mimepull, SQL2O, Hikary, mySQL.
Source code can be found at https://github.com/rembli/documents
Thanks to Federico Tomassetti for his great tutorials at https://sparktutorials.github.io.
